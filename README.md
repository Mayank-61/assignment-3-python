# assignment-3-python
# Assignment 3 - Functions & Modules in Python

## Task 1: Factorial Using a Function
This program defines a function `factorial(n)` that calculates the factorial
of a given non-negative integer using a loop. The user is prompted to enter
a number, and the factorial is displayed.

File: task1_factorial.py

## Task 2: Using the Math Module
This program asks the user for a number and uses Python's `math` module to
calculate:
- Square root of the number
- Natural logarithm (log base e)
- Sine of the number (in radians)

File: task2_mathmodule.py

## How to Run
1. Make sure Python is installed.
2. Run the files using:
   python task1_factorial.py
   python task2_mathmodule.py
